# Week 5: Business Analytics Case Study

---

## Business Problem
Why are customers leaving Telco, and which factors contribute most to customer churn?

## Dataset
**Telco Customer Churn Dataset** — [Kaggle](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
- 7,043 records | 21 features | Target variable: `Churn` (Yes/No)

## Tools Used
- Python (Pandas, Matplotlib, Seaborn) — Jupyter Notebook
- Microsoft Word — Business Report
- Microsoft PowerPoint — Presentation

## Key Findings
| Driver | High-Risk Segment | Churn Rate |
|---|---|---|
| Contract Type | Month-to-month | 42.71% |
| Payment Method | Electronic check | 45.29% |
| Internet Service | Fiber optic | 41.89% |
| Tech Support | No support | 41.65% |
| Tenure | Avg. 18 months (churned) vs. 37.7 months (retained) | — |

## Top Recommendations
1. Incentivize customers to switch to long-term contracts
2. Encourage migration to automatic payment methods
3. Improve tech support accessibility and visibility
4. Reassess fiber optic pricing and value proposition
5. Launch a new-customer retention program for the first 12 months
